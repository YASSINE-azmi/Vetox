# 🚗 Vetox - Agence de Location de Voitures

## 📝 À propos du projet
**Vetox** est une plateforme web moderne conçue pour une agence de location de véhicules premium basée dans la région de Souss-Massa (Agadir / Ait Melloul). Le projet se distingue par une interface utilisateur (UI) élégante, pensée pour offrir la meilleure expérience utilisateur (UX) possible.

## ✨ Fonctionnalités Principales
* **Design UI/UX Premium :** Maquettes haute fidélité réalisées sur Figma avant le développement.
* **Mode Sombre (Dark Mode) :** Support natif avec transition fluide entre le thème clair et sombre.
* **Catalogue Dynamique :** Affichage des véhicules sous forme de grille (CSS Grid) avec détails des prix par jour.
* **Design Réactif (Responsive) :** Interface parfaitement adaptée aux smartphones, tablettes et ordinateurs (Mobile-First).
* **Formulaires Interactifs :** Simulation de système de réservation et page de contact.

## 🛠️ Technologies Utilisées
* **Design & Prototypage :** Figma
* **Frontend :** HTML5, CSS3 (Flexbox, Grid, CSS Variables)
* **Logique & Interactivité :** JavaScript (Vanilla)
* **Contrôle de version :** Git & GitHub
